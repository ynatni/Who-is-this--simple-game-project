# Who-is-this--simple-game-project
Who Is This? adalah proyek game sederhana yang dibuat oleh Kelompok 7 sebagai bagian dari Ujian Tengah Semester (UTS) Semester 2.

Proyek ini menggabungkan konsep dari beberapa mata kuliah:
1. Algoritma dan Pemrograman
2. Praktikum Algoritma dan Pemrograman
3. Struktur Data
4. Praktikum Struktur Data

Deskripsi Game: 
Kamu berperan sebagai seorang detektif di Greyford City yang ditugaskan untuk menyelidiki kasus pembunuhan.
Korban bernama Sheryl Iris (19) ditemukan tewas di apartemennya.
Terdapat 5 orang tersangka yang harus kamu analisis berdasarkan data dan bukti yang tersedia.

Tujuan Game:
Untuk menyelesaikan game, pemain harus:
1. Menganalisis data dan mengisi checklist investigasi dengan benar
2. Memilih tersangka yang paling sesuai dengan bukti

Apakah kamu bisa menemukan pelaku sebenarnya dan menegakkan keadilan?

Cara menjalankan Game:
Opsi 1: File executable
1. Unduh file WhoIsThis.exe
2. Pastikan dependency sudah tersedia (jika diperlukan)
3. Jalankan file tersebut

Opsi 2: Melalui source code 
1. Install Raylib dari website resmi: https://www.raylib.com/
   atau melalui GitHub: https://github.com/raysan5/raylib
2. Pastikan versi Raylib sesuai dengan compiler yang digunakan
3. Extract dan setup library
4. Jika menggunakan VS Code: Tambahkan path include Raylib ke file c_cpp_properties.json
